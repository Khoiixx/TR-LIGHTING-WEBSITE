# TR Lighting Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Black (#0a0a0a) - Main background
- **Secondary**: Dark Gray (#1a1a1a) - Card backgrounds
- **Accent**: Neon Purple (#9333ea) - Glowing effects, CTAs
- **Highlight**: Bright Purple (#a855f7) - Hover states
- **Text**: Light Gray (#e5e5e5) - Primary text
- **Muted**: Gray (#9ca3af) - Secondary text

### Typography
- **Display Font**: "Orbitron" - Futuristic sans-serif for headings
- **Body Font**: "Inter" - Clean, modern sans-serif for content
- **Accent Font**: "JetBrains Mono" - Monospace for technical elements

### Visual Language
- **Minimalist Futuristic**: Clean lines with high-tech aesthetics
- **Glowing Accents**: Subtle neon effects on interactive elements
- **Geometric Patterns**: Grid overlays and angular shapes
- **Depth & Layering**: Subtle shadows and elevation effects
- **Smooth Animations**: Fluid transitions and micro-interactions

## Visual Effects

### Used Libraries
- **Anime.js**: Button animations, page transitions, text effects
- **Typed.js**: Typewriter animation for hero text
- **Splitting.js**: Character-by-character text animations
- **p5.js**: Background particle effects and gradient flows
- **Pixi.js**: Advanced visual effects for hero background
- **ECharts.js**: Data visualization (if needed)

### Animation Effects
- **Typewriter Text**: "The Future of Lighting for Events" with cursor
- **Glowing Buttons**: Purple neon glow on hover with pulse effect
- **Card Lift**: Product cards elevate with shadow on hover
- **Gradient Flow**: Animated background with purple/blue gradients
- **Particle System**: Subtle floating particles for atmosphere
- **Smooth Transitions**: Page navigation with fade effects

### Styling Approach
- **Glass Morphism**: Semi-transparent cards with backdrop blur
- **Neon Glow**: CSS box-shadow and text-shadow for lighting effects
- **Grid Layout**: CSS Grid for responsive product layouts
- **Flexbox**: Navigation and component alignment
- **Custom Properties**: CSS variables for consistent theming
